# To Do List Uygulaması

Basit ama modern bir **To Do List (Görev Listesi) web uygulaması**.
Bu uygulama sayesinde görev ekleyebilir, silebilir, düzenleyebilir ve filtreleyebilirsin.

Proje **HTML, CSS ve JavaScript** kullanılarak geliştirilmiştir ve tamamen tarayıcı üzerinde çalışır.

---

##  Önizleme

![To Do List Preview](https://github.com/BaranDogrular/responsive-landing-page/blob/f74f53fe2977bb339a9ee0431b4f116eab7a4795/screenshot..png)

##  Özellikler

*  Görev ekleme
*  Görev silme
*  Görev düzenleme 
*  Görevi tamamlandı olarak işaretleme
*  Görev filtreleme
*  Dark Mode
*  Görev sayacı
*  Tüm görevleri temizleme
*  LocalStorage ile görevleri kaydetme

---

##  Kullanılan Teknolojiler

* HTML5
* CSS3
* JavaScript 

---

##  Projenin Amacı

Bu proje, **JavaScript DOM manipülasyonu, LocalStorage kullanımı ve temel kullanıcı arayüzü geliştirme** konularında pratik yapmak amacıyla geliştirilmiştir.
